# Chunk Mate - Frontend

This is the frontend application for **Chunk Mate**, a tool designed to upload Markdown documents, process them into context-rich chunks, and manage extracted links and tables.

Built with **React.js**.

---

## 🚀 Features

- Upload `.md` documents
- Automatically chunk paragraphs, tables, and headings
- Extract links into a reference table
- View and interact with uploaded document history
- Responsive UI design
